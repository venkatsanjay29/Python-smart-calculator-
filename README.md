# Python-smart-calculator-

A simple calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu
- Handles invalid input

## Technologies Used
- Python 3

## How to Run
1. Install Python 3.
2. Download `calculator.py`.
3. Run:
   ```bash
   python calculator.py
   ```
